# 📜 測驗腳本：G-MBTI 戀愛地城靈魂審判 (v1.0)

## 1. 測驗說明
- **目標維度**：E/I (能量來源)、S/N (感知方式)、T/F (判斷標準)、J/P (生活態度)。
- **計分方式**：每題二選一，對應維度加 1 分，最終統計各維度最高分者為結果。

---

## 2. 測驗題目 (共 20 題)

### 第一階段：能量來源 (E vs I)
**情境：當你在冒險者公會的酒館休息時...**

1. **看到一群高等精靈（異性）在慶功，你的直覺反應是：**
   - (E) 主動拿著酒杯過去，試圖加入話題，看看有沒有組隊機會。
   - (I) 默默移動到角落陰影處，低頭擦拭自己的破舊匕首，祈禱別被注意到。

2. **你收到一個不知名的魔法卷軸（Threads 標記或私訊），你會：**
   - (E) 立刻回覆並開始興奮地與對方交換情報，感受社交的熱度。
   - (I) 盯著卷軸看很久，思考了三小時後還是決定當作沒看到。

3. **經歷了一整天慘烈的地城戰鬥（社交應酬），你如何恢復魔力？**
   - (E) 跟隊友繼續去續攤酒館，在大笑與喧鬧中找回動力。
   - (I) 躲回自己安靜的洞穴，一個人發呆或整理裝備。

4. **當你在公會看板尋找任務（滑交友軟體）時：**
   - (E) 只要看起來不錯就立刻送出組隊請求，數量就是力量。
   - (I) 仔細閱讀每個人的背景故事，猶豫再三才敢投遞一張匿名申請。

5. **關於「孤獨」這件事，你的看法是：**
   - (E) 簡直是慢性的毒素傷害，沒人一起刷怪的生活毫無意義。
   - (I) 是我最好的防禦護盾，在人群中反而會持續掉血。

### 第二階段：感知方式 (S vs N)
**情境：關於你心目中的「命定隊友」...**

6. **第一次與潛在隊友（約會對象）吃飯，你最在意的是：**
   - (S) 點餐的價格、對方是否有確實 AA 制、食物的 CP 值。
   - (N) 聊天時有沒有心靈感應，以及對方是否符合你腦中的完美英雄形象。

7. **當對方送你一個「守護墜飾」作為禮物，你首先想到：**
   - (S) 這個墜飾的材質是什麼？市場價值多少金幣？實用嗎？
   - (N) 這代表他想守護我一生嗎？這是命運的契約嗎？（開始腦補結婚畫面）

8. **你在地城入口遇到一位受傷的精靈，你會：**
   - (S) 觀察對方的傷勢，評估治療藥水的成本與救人後的報酬。
   - (N) 感受到這是一場史詩冒險的開端，認定自己是救贖對方的英雄。

9. **關於未來，你的規劃通常是：**
   - (S) 精確的存款清單、保險規劃，以及確保明天有飯吃。
   - (N) 幻想有一天能屠龍成功，並在城堡裡過著幸福生活（雖然目前 Lv.1）。

10. **你如何解讀對方的訊息（暗示）？**
    - (S) 字面上說什麼就是什麼，對隱喻和潛台詞感到頭痛。
    - (N) 會反覆推敲文字背後的「魔力波動」，試圖解讀出一百種可能。

### 第三階段：判斷標準 (T vs F)
**情境：當隊伍發生衝突或對方受傷時...**

11. **當隊友（伴侶）在地城裡犯錯導致滅團，你的第一反應：**
    - (T) 指出邏輯上的失誤，並開始說明正確的操作流程（開始說教）。
    - (F) 關心對方有沒有受驚嚇，給予一個溫暖的安慰與擁抱。

12. **如果你需要拒絕一個不感興趣的組隊請求，你會：**
    - (T) 直接列出雙方戰力不匹配的事實，理智地拒絕。
    - (F) 擔心傷害對方的自尊心，編造一個委婉的藉口或裝死。

13. **當你心情不好時，你希望對方施放什麼技能？**
    - (T) 給予實質的建議，分析現況並提供解決方案。
    - (F) 不需要解決方案，只需要陪我一起罵地城裡的魔物。

14. **在一段關係中，你認為最重要的是：**
    - (T) 公平、互惠、且彼此符合邏輯的契約。
    - (F) 共鳴、和諧、以及那份無可取代的情緒價值。

15. **當別人說你「很奇怪」時：**
    - (T) 思考這句話的邏輯是否成立，並試圖辯論。
    - (F) 感到極度受挫，心碎了一地，想躲回洞穴哭泣。

### 第四階段：生活態度 (J vs P)
**情境：關於冒險的規劃與執行...**

16. **週末要去刷一個新的副本，你的風格是：**
    - (J) 事前查好所有攻略、準備好對應抗性的藥水、訂好集合時間。
    - (P) 到門口再說，看當下的心情決定要不要進去，隨遇而安。

17. **關於你的冒險日誌（行事曆）：**
    - (J) 條列式記錄每一項行程，沒按計畫走會感到焦慮。
    - (P) 雜亂無章，充滿隨手塗鴉，想起來才記一筆。

18. **當你答應了對方的組隊邀請，但臨時有更強的獵物出現：**
    - (J) 堅持原本的承諾，認為誠信是冒險者的基本。
    - (P) 可能會失約或遲到，因為「更有趣的事情」發生了。

19. **關於你的「老媽（長老）」對你冒險的干預：**
    - (J) 習慣詢問長老的意見，甚至讓長老決定我要穿哪套防具。
    - (P) 追求自由，長老越反對我越想去闖蕩。

20. **你的房間（哥布林巢穴）通常是：**
    - (J) 物品分類整齊，甚至連戰利品都要按顏色擺放。
    - (P) 像被巨龍踩過一樣，但宣稱「這是一種亂中有序的藝術」。
